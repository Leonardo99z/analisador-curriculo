# Analisador de Currículo com IA

Aplicação que analisa currículos usando Inteligência Artificial e retorna score de compatibilidade, pontos fortes, lacunas e sugestões de melhoria.

## 🔗 Demo
https://ai-cv-boost-11.lovable.app

## 🛠️ Tecnologias
- Python
- FastAPI
- Groq (LLaMA 3.3)
- React (Lovable)
- Render (deploy)

## ⚙️ Como funciona
1. Usuário cola o currículo e a descrição da vaga
2. Backend Python chama a IA via Groq
3. IA retorna score, pontos fortes, lacunas e sugestões
